# PetsGPS — Vite + React prototype

Landing prototype reproduciendo la maqueta entregada.

Run locally (PowerShell):

```powershell
npm install
npm run dev
```

Abre http://localhost:5173 en tu navegador.

Estructura principal:
- `src/` - código fuente
- `src/components` - componentes React
- `src/assets` - imágenes SVG usadas en la maqueta
- `src/styles/global.css` - estilos globales

Sugerencias: reemplaza los SVG en `src/assets` por fotos reales si las tienes.
