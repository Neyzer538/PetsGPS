# Product Images Directory

Place product detail images here with the following naming convention:

- `collar-detail-1.jpg` - First detail image for collar product
- `collar-detail-2.jpg` - Second detail image for collar product
- `rastreador-detail-1.jpg` - First detail image for rastreador product
- `rastreador-detail-2.jpg` - Second detail image for rastreador product
- `localiza-detail-1.jpg` - First detail image for localiza product

The application will fallback to existing images if these are not found.
